# Arduino workshop

In this workshop, we will learn how to use an Arduino.

## Goal

The goal of this workshop is that you are able to use 
booklets, laptops and Arduinos to create a minimal electronic machine.

- Date: Saturday August 8th 2026
- Time: 12:00 (sharp!) to 14:00 (see [schedule](schedule.md) for detailed schedule)
- [Target audience: any Uppsala Makerspace member](faq.md#at-what-level-is-the-workshop-taught)
- [Bring with you: nothing. Optional/ideally: a laptop and/or a T-shirt](faq.md#what-do-i-need-to-bring)
